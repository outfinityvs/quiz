Privacy by Architecture

Questions and answers are processed exclusively in your browser. Outfinity Quiz does not use accounts, cookies, analytics, pixels, fingerprinting, or external scripts.

What We Do Not Receive

When you take a quiz, your answers, scores, and profile exist only in the memory of the current page. Closing or reloading the page erases everything. We do not receive any of this information.

The quiz pages enforce this through a Content Security Policy directive (connect-src 'none') that prevents JavaScript from making any network requests after the page loads. This is a technical guarantee, not a policy promise.

Shareable Links

A shareable link is created only when you explicitly request it. The link encodes your aggregated result (scores on each dimension, archetype) in the URL fragment — the part after the # symbol.

The URL fragment is processed by your browser and is not sent to the server when the page is requested. This means the hosting provider does not see your result, even in server logs.

The shareable payload never contains individual answers, names, email addresses, completion time, professional domains, or unique identifiers.

External Links

Some quiz results include an optional next step pointing to an Outfinity initiative, a venture, or a partner. Clicking these links opens an external page in a new tab with rel="noopener noreferrer". No quiz data is transmitted through these links.

If you choose to open Explorer Circle (hosted on Substack), Substack's own privacy policy applies. Your quiz result is not transmitted automatically.

Hosting

The hosting provider may retain standard technical access logs for page requests. These logs contain the URL path and standard HTTP headers, but not the URL fragment or any quiz answers.

Local Storage

By default, quiz state exists only in page memory. If a quiz offers the option to save progress locally, this requires your explicit action. You can always delete all locally stored data from the same interface.
