# Trust Architecture

## Introduction

The Trust Architecture quiz maps how the user builds, verifies, and loses trust across different contexts.

## Core Content

### Source and Licensing

Original Outfinity instrument.

Source: Original
License: CC BY 4.0

### Instrument Design

**Question Count:** 10 scenarios
**Response Format:** Single choice from 5 options (one per trust mechanism)
**Duration:** 3 minutes
**Scoring:** Each option maps to one trust mechanism

### Dimensions (Trust Mechanisms)

1. **Relational** - Trust through face-to-face connection, personal bonds, loyalty
2. **Contractual** - Trust through written agreements, legal frameworks, explicit terms
3. **Reputational** - Trust through track record, references, past performance
4. **Procedural** - Trust through transparent process, audits, verification
5. **Communal** - Trust through shared governance, collective ownership, mutual accountability

### Viral Mechanics

- **Trust blueprint comparison:** Overlay two trust architectures
- **Partnership compatibility:** "Your trust styles align/diverge in these ways"
- **Challenge:** "How do you build trust?" - share quiz URL

### Result Interpretation

Shows primary and secondary trust mechanisms with interpretive text describing how the user builds and verifies trust.
