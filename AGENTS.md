# AGENTS.md

## Scope

Outfinity Quiz is a static subsite hosting privacy-first self-knowledge questionnaires. Every quiz runs entirely in the browser. No answers, scores, or profiles are transmitted to any server. The subsite is deployed to `quiz.outfinity.ch` from the `docs/` directory.

## Mandatory Reading Order

1. `docs/specs/DS000-vision.md` — product vision, user contract, and phase plan.
2. `docs/specs/DS001-coding-style.md` — coding style, module structure, and test organization. This is the coding-style authority.
3. `docs/specs/DS007-site-structure.md` — routes, navigation, and catalog organization.
4. `docs/specs/DS008-privacy-security.md` — privacy architecture and security headers.
5. The per-quiz DS files (DS009 onward) before modifying any quiz.
6. The per-module DS files (DS002–DS006) before modifying any engine module.

## Repository Rules

- All documentation, specifications, and comments must be written in English.
- DS specifications under `docs/specs/` are the source of truth for documented behavior.
- When source code changes, both the HTML documentation and the DS specifications must be updated to reflect the change.
- DS numbering must remain contiguous with no gaps. The next new DS after DS056 is DS057.
- `Decisions & Questions` uses numbered question subchapters. Rationale lives in the affected DS files.
- No external dependencies: no React, Vue, Tailwind, CDN libraries, external fonts, analytics, or social SDKs.
- All quiz logic runs client-side. No backend, no API calls, no `fetch` after page load.
- Quiz data (questions, scoring, interpretation) is defined in JSON files. The engine is generic and reusable.
- The `docs/dev/` subfolder contains developer-facing technical documentation. The quiz site itself occupies the root of `docs/`.

## Runtime Defaults

- The subsite is served as static files from `docs/`.
- No build step is required for the quiz site itself.
- The specs matrix is generated by running `node .agents/skills/gamp-specs/scripts/generate_specs_matrix.mjs`.
- Documentation link verification: `node .agents/skills/gamp-specs/scripts/verify_docs_links.mjs`.
- Static site verification: `node .agents/skills/gamp-specs/scripts/verify_static_site.js docs/`.
